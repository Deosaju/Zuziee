
## Run Locally

Clone the project

```bash
  git clone https://github.com/Deosaju/Zuziee
```

Go to the project directory

```bash
  cd Zuziee
```


## Installation

cd server

```bash
  npm i
  npm run start
```

cd zuzie

    
```bash
  npm i
  npm run dev
```

`GITHUB_SECRET`

`GITHUB_ID`

`REDIRECT_URL`
